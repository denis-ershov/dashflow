# Extension Icons

Add the following PNG icons for Chrome Web Store submission:

| File       | Size    | Purpose                          |
|------------|---------|----------------------------------|
| icon-16.png | 16×16   | Favicon for extension pages      |
| icon-48.png | 48×48   | Extensions management page       |
| icon-128.png| 128×128 | **Required** – Install & Web Store |

WXT auto-discovers these files. See `docs/WEB_STORE_SUBMISSION.md` for design tips.
