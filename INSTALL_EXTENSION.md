# Сборка и установка расширения для Google Chrome

## Требования

| Инструмент | Минимальная версия |
|------------|-------------------|
| Node.js    | 18+               |
| pnpm       | 9+                |

Проверить наличие:
```bash
node --version
pnpm --version
```

---

## 1. Установить зависимости

Из корня репозитория:
```bash
pnpm install
```

---

## 2. Собрать расширение

```bash
pnpm --filter @workspace/newtab-dashboard build
```

Либо перейдите в папку расширения и запустите оттуда:
```bash
cd artifacts/newtab-dashboard
pnpm build
```

После успешной сборки появится папка:
```
artifacts/newtab-dashboard/.output/chrome-mv3/
```

Это и есть готовое расширение.

---

## 3. Установить в Google Chrome

1. Откройте Chrome и перейдите по адресу:
   ```
   chrome://extensions
   ```

2. Включите **«Режим разработчика»** (переключатель в правом верхнем углу).

3. Нажмите кнопку **«Загрузить распакованное расширение»**.

4. В открывшемся диалоге выберите папку:
   ```
   artifacts/newtab-dashboard/.output/chrome-mv3
   ```

5. Нажмите **«Выбрать папку»**.

6. Откройте новую вкладку — вместо стандартной страницы появится ваш дашборд.

---

## 4. Создать ZIP-архив для дистрибуции (опционально)

Если нужно поделиться расширением или загрузить его в Chrome Web Store:
```bash
pnpm --filter @workspace/newtab-dashboard zip
```

ZIP-файл появится в папке:
```
artifacts/newtab-dashboard/.output/
```

---

## 5. Разработка с горячей перезагрузкой

Для разработки с автоматической перезагрузкой расширения в браузере:
```bash
pnpm --filter @workspace/newtab-dashboard dev
```

WXT откроет отдельный экземпляр Chrome с уже загруженным расширением и будет следить за изменениями файлов.

> Эта команда запускает **WXT** (режим расширения), а не Vite-сервер для предпросмотра в Replit.

---

## Разрешения расширения

Расширение запрашивает следующие разрешения Chrome:

| Разрешение   | Зачем нужно                                      |
|-------------|--------------------------------------------------|
| `storage`   | Сохранение виджетов, настроек, заметок и привычек |
| `bookmarks` | Доступ к закладкам браузера                      |
| `geolocation` | Определение города для виджета погоды          |

---

## Структура сборки

```
.output/chrome-mv3/
├── manifest.json          ← Манифест расширения (MV3)
├── newtab.html            ← Страница новой вкладки
├── popup.html             ← Всплывающее окно расширения
├── assets/                ← JS, CSS, шрифты
├── icon-16.png
├── icon-32.png
├── icon-48.png
└── icon-128.png
```

---

## Обновление после изменений в коде

После правок в исходниках нужно пересобрать расширение и обновить его в Chrome:

```bash
pnpm --filter @workspace/newtab-dashboard build
```

Затем на странице `chrome://extensions` нажмите иконку **↺ (обновить)** напротив расширения **Personal Start Page**.
