# 🚀 DashFlow

**Создай свой поток продуктивности**

Современное Chrome расширение, которое заменяет стандартную стартовую страницу на персонализированную приборную панель с виджетами.

## ✨ Особенности

- 🎨 **Современный дизайн** с поддержкой светлой/темной темы
- 🧩 **Drag & Drop виджеты** - перетаскивайте и изменяйте размер
- 🌍 **Мультиязычность** - русский и английский языки
- ⚡ **Высокая производительность** на React 19 + Vite 6
- 📱 **Адаптивный дизайн** для разных разрешений экрана
- 💾 **Синхронизация настроек** между устройствами
- 🎛️ **Гибкая настройка** через CSS и темы

## 🔧 Виджеты

### 📊 Информационные
- **Погода** - текущая погода и прогноз с геолокацией
- **Часы** - цифровые/аналоговые часы с часовыми поясами
- **Курс валют** - актуальные курсы валют с конвертером
- **Криптовалюты** - курсы Bitcoin, Ethereum и других монет
- **RSS ленты** - новости из любых RSS источников

### 📋 Продуктивность
- **To-Do список** - задачи с приоритетами и дедлайнами
- **Календарь** - синхронизация с Google Calendar
- **Заметки** - цветные стикеры с поиском
- **Канбан доска** - управление проектами
- **Текстовый редактор** - Rich text заметки

### 🔗 Инструменты
- **Поисковая строка** - поиск через разные системы
- **Закладки** - управление закладками браузера
- **iFrame виджет** - встраивание любых веб-страниц
- **Аудио плеер** - интеграция со Spotify/SoundCloud

## 🛠️ Технологии

- **WXT Framework** v0.20.8 - современный фреймворк для расширений
- **React 19** - последняя версия React с новыми возможностями
- **TypeScript** - строгая типизация для надежности

## 🚀 Установка и запуск

### Предварительные требования
- Node.js 18+ 
- npm или yarn

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```

### Сборка для продакшена
```bash
npm run build
```

### Установка в Chrome

1. Откройте Chrome и перейдите в `chrome://extensions/`
2. Включите "Режим разработчика"
3. Нажмите "Загрузить распакованное расширение"
4. Выберите папку `.output/chrome-mv3` (после сборки)
5. Расширение будет установлено и активировано

## 🎨 Кастомизация

### Темы
Расширение поддерживает три режима темы:
- **Светлая** - классический светлый интерфейс
- **Темная** - современный темный режим
- **Авто** - следует системной теме

### Цветовая схема

#### Темная тема
- Основной фон: `#12232E`
- Акцент: `#007CC7`
- Вторичный акцент: `#4DA8DA`
- Тени/границы: `#203647`

#### Светлая тема  
- Основной фон: `#F5FAFD`
- Текст: `#1A3A4A`
- Акцент: `#007CC7`
- Вторичный акцент: `#4DA8DA`
- Тени/границы: `#9BBECF`

### Пользовательский CSS
Добавляйте собственные стили через панель настроек:
```css
.widget-container {
  border-radius: 20px;
  box-shadow: 0 8px 32px rgba(0,0,0,0.1);
}

.glass-morphism {
  backdrop-filter: blur(20px);
  background: rgba(255,255,255,0.1);
}
```

## 🤝 Разработка

### Добавление нового виджета

1. **Создайте тип виджета** в `src/types/widget.ts`:
```typescript
export type WidgetType = 'weather' | 'clock' | 'my-widget';
```

2. **Создайте компонент** в `src/components/widgets/MyWidget.tsx`:
```typescript
import React from 'react';
import { WidgetConfig } from '../../types/widget';

interface MyWidgetProps {
  config: WidgetConfig;
}

const MyWidget: React.FC<MyWidgetProps> = ({ config }) => {
  return <div>Мой виджет</div>;
};

export default MyWidget;
```

3. **Добавьте в Dashboard.tsx** рендеринг нового виджета

4. **Добавьте переводы** в `src/locales/*/widgets.json`

### Создание темы
1. Добавьте цвета в `tailwind.config.js`
2. Создайте CSS переменные в `globals.css`
3. Добавьте тему в `ThemeStore`
